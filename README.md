# Macaroons Landing
## Описание проекта 
- Лендинг кондитерской «Macaroon», демонстрирующий навыки работы с Angular, реактивными формами и управлением состоянием. Проект не требует бэкенд-сервера: данные о товарах и блоках страницы хранятся в TypeScript-массивах и динамически рендерятся в шаблонах.
  
## Ключевые особенности реализации
- Имитация бэкенда: Каталог товаров и контент секций вынесены в типизированные TypeScript-интерфейсы и массивы. Это позволяет легко менять ассортимент без правки HTML-шаблонов.
- Псевдокорзина и подсчёт стоимости: При клике на кнопку «Заказать» в карточке товара позиция добавляется в корзину, а общая сумма и количество товаров обновляются реактивно. Реализован сервис `CartCountService`, который синхронизирует счётчик товаров между разными частями приложения.
- Динамическая подстановка в форму: При клике на кнопку «Заказать» в карточке товара происходит автоматический скролл к секции заказа и реактивное обновление поля «Ваш выбор» в форме (FormControl).
- Имитация отправки формы: При клике «Оформить заказ» происходит сброс формы, обнуление счетчика корзины и уведомление, что заказ  отправлен.
- Адаптивная верстка: Проект корректно отображается на мобильных устройствах и десктопах.

## Стек технологий

- Frontend: Angular 21, TypeScript.
- Стили: SCSS.
- Навигация: Нативный scrollIntoView для плавного скролла.
## Запуск проекта
- `npm i` && `ng serve --open`
